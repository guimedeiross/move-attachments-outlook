1º - Instalar o python

2º - Abra o cmd e execute o seguinte comando "pip install pywin32" sem as aspas.

3º - crie a pasta "moverArquivos" no C:

4º - abra o arquivo mover.py e edite o caminho para onde você quer salvar o anexo.

5º - Quando criar o script nas regras, coloque para iniciar um aplicativo e coloque o "executar.vbs" pra iniciar.

Ps: Testado em um Windows 7 Ultimate x64 usando Pacote Office 2010 Standard em uma conta POP.
